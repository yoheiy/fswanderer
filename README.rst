README
======

fswanderer is a native win32 filemanager.

key binding
-----------

- j move cursor down
- k move cursor up
- m mark a file
- o open directory
- u walk toward root
- V toggle preview
- x delete marked backup file (*~)
- q quit from fswanderer

and more...

install
-------

1. install MinGW.
2. make.  (fix CC in Makefile if necessary)

run
---

1. install Wine. (first time)
2. wine ./fswanderer
